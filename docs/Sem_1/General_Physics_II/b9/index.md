## Current and Drift Speed
$\displaystyle I_{avg} = \frac{\Delta Q}{\Delta t} = \frac{n.q}{\Delta t} = \frac{\frac{n}{V}.V.q}{\Delta t} = n.A.q.v$ 
* **n is the number of mobile charge carriers per unit volume**
