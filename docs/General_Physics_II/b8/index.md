## Electric Dipole
<img src = "P5KFKPu.png" width = 400 height = 300>

* $F = qE$
* $\vec F_{net} = 0$
* $\tau_{net} = p.E.sin\theta$
* **Electric dipole moment:** $\vec p = q.\vec d$
* **Torque on the dipole:** $\vec \tau = \vec p. \vec E$
* **Potential energy:** $U = -\vec p. \vec E$
